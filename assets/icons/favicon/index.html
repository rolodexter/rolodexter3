ROLODEXTER Favicon Requirements

Required Favicon Files

  • favicon.ico

    Multi-size ICO file containing:

    • 16×16 pixels
    • 32×32 pixels

  • favicon-32x32.png

    Standard favicon for modern browsers

  • apple-touch-icon.png

    180×180 pixels for iOS devices

  • android-chrome-192x192.png

    192×192 pixels for Android devices

  • android-chrome-512x512.png

    512×512 pixels for PWA requirements
