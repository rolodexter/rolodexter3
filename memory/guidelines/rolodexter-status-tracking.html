Automation with GitHub Actions

Status updates are automated using GitHub Actions to ensure consistency:

      - name: Update rolodexter's Status in HTML Files
        run: |
          TIMESTAMP=$(date -u +"%Y-%m-%d %H:%M:%S UTC")
          for file in $(find . -name "*.html"); do
            if grep -q "class=\"status-block\"" "$file"; then
              sed -i "s|Last Updated:.*UTC|Last Updated: $TIMESTAMP UTC|" "$file"
            fi
          done
